### Church History App

This is my personal Church History archive. As I study I will collect data about historical figures and upload my database with the information. 

#### Basic outline

I have a postgresdb which I access via nocodb. This webapp collects the data from the postgresdb and displays it in a better way for me have an overview of the historical figures I am studying. I can also then share this with others. Mainly python files and the frontend ui is html.
